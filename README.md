## Pasos para Ejecutar Ejemplos

### Paso 1: Clonar proyecto

```shell

```

### Paso 2: Crear entorno

```shell
virtualenv env_smma -p 3.8
```

### Paso 3: Instalar Dependencias

```shell
pip install -r requirements.txt
```
